# Axis_AQS_Data_Visualization
A complete Docker Compose MING stack (Mosquitto MQTT, InfluxDB, Node-RED, Grafana) deployable via Portainer for ingesting Axis D6310 Air Quality Sensor data over MQTT and visualizing in Grafana. Includes pattern detection for occupancy (high CO2) and cleaning (PM spikes), Node-RED flows, provisioning, and dashboards.
